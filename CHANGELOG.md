## 0.0.1

* Versión inicial del sistema de diseño Pablito DS
* Implementación completa de Atomic Design con 5 niveles:
  - **Átomos**: Botones, inputs, tipografía, iconos, badges, divisores
  - **Moléculas**: Cards, formularios, barra de búsqueda, items de lista, avatares, alertas
  - **Organismos**: Headers, footers, tablas de datos, navegación, formularios complejos
  - **Plantillas**: BaseLayout, DashboardLayout, AuthLayout
  - **Páginas**: LoginPage, DashboardPage
* Sistema de tokens de diseño centralizado
* Tema Material Design 3
* Aplicación showcase completa con ejemplos de todos los componentes
* Documentación detallada en README
